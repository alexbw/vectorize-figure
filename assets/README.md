# Reference Images

Every image in `reference/` and `full-reference/` was generated by an AI image
model (Nano Banana Pro / gpt-image) prompted to produce publication-style
scientific figures. None of them come from a published paper. The "data" they
depict is hallucinated by construction — which is exactly the problem this
project exists to fix.

These images are the test fixtures for the validation suite. Do not resize,
recompress, or otherwise modify them: the JSON specs in `outputs/` store
source-calibrated pixel coordinates measured against these exact files.
